# Electronics Distribution Analytics Dashboard

## Project Overview

Built an interactive Power BI dashboard to analyze sales performance, profitability, shipment trends, product performance, and regional distribution across an electronics distribution business.

The dashboard enables users to explore key business metrics using dynamic filters, bookmarks, field parameters, custom DAX measures, and interactive visualizations.

---

## Tools Used

- Power BI
- Power Query
- DAX
- CSV Data Sources
- Data Modeling (Star Schema)

---
